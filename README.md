# 📊 PhonePe Digital Payments Analysis (India)

## 🚀 Project Overview

This project analyzes digital payment trends using transaction and demographic data inspired by PhonePe datasets. The goal is to uncover insights related to transaction behavior, user growth, device usage, and demographic influence across Indian states and districts.

---

## 🎯 Objectives

* Analyze transaction trends across states and time (year & quarter)
* Understand user behavior and app engagement
* Identify top-performing states and districts
* Explore the impact of demographics (population, density)
* Perform advanced data analysis and correlations
* Ensure data consistency between state and district levels

---

## 📂 Dataset Description

The project uses multiple datasets:

1. **State Transactions & Users**
2. **State Transaction Split**
3. **State Device Data**
4. **District Transactions & Users**
5. **District Demographics**

Each dataset includes information such as:

* Transactions count & amount
* Average Transaction Value (ATV)
* Registered users & app opens
* Device brand usage
* Population, area, and density

## 📈 Key Analysis Performed

### 🔹 Data Cleaning & Preparation

* Handled missing values
* Checked data types
* Ensured data consistency

### 🔹 Exploratory Data Analysis (EDA)

* Transaction trends over time
* Top & bottom states by transaction volume
* Most common transaction types
* Device brand analysis

### 🔹 Advanced Analysis

* User-to-population ratio
* Correlation between population density & transactions
* Average transaction amount per user
* District-level insights

### 🔹 Data Validation

* Compared state-level vs district-level aggregates
* Identified discrepancies in data

---

## 📊 Visualizations

* Line plots for transaction trends
* Bar charts for state comparisons
* Scatter plots for correlation analysis
* Pie charts for transaction type distribution

---

## 🔍 Key Insights

* States with higher population density tend to show higher transaction volumes
* Digital adoption is strongly linked with urban districts
* Certain device brands dominate user share across states
* App engagement (app opens) correlates with transaction activity

---

## 💡 Business Recommendations

* Focus on low-performing states for user acquisition
* Optimize app performance for dominant device brands
* Increase awareness campaigns in low-density regions
* Enhance user engagement to boost transactions
